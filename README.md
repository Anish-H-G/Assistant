# Assistant
Developed an AI-powered Speech-to-Text and Text-to-Speech assistant using Azure Cognitive Services to enhance accessibility and user interaction.

Implemented real-time speech recognition to transcribe audio into text with high accuracy.

Integrated Text-to-Speech functionality to convert text input into natural-sounding speech output.

Designed and deployed a web-based interface for seamless user experience.

Optimized API usage and response times for efficient real-time communication.
